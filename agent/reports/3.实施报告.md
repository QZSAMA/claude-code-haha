## 4. 遇到的问题及解决方案

### 问题1: Python路径导入

**问题**: 测试运行时无法找到 `claude_agent` 包

**原因**: 测试文件在 `agent/tests/`，需要将上级目录加入sys.path

**解决方案**: 在测试文件开头添加:
```python
import sys
import os
sys.path.insert(0, os.path.join(os.path.dirname(__file__), '..'))
```

**状态**: ✅ 已解决

### 问题2: 缺少import

**问题**: 部分工具文件缺少 `Optional` 或 `BaseModel` import

**原因**: 复制代码片段时遗漏了import

**解决方案**: 检查每个文件，补全所有import

**状态**: ✅ 已解决

### 问题3: 测试断言错误

**问题**: `test_needs_compaction` 断言失败，800 tokens 在 0.8 × 1000 = 800 阈值时不触发压缩，需要严格大于

**原因**: 边界条件理解错误，原代码是 `estimated_tokens > max_tokens × threshold`

**解决方案**: 修改测试用例，使用 801 触发压缩，800 不触发

**状态**: ✅ 已解决

### 问题4: 网络问题安装依赖

**问题**: 最初安装anthropic时网络重试多次

**原因**: 网络连接不稳定

**解决方案**: pip会自动重试，最终成功安装

**状态**: ✅ 已解决

### 问题5: PowerShell路径分隔符

**问题**: Windows PowerShell中路径分隔符问题

**解决方案**: 使用Python `os.path` 模块处理路径，自动适配操作系统

**状态**: ✅ 不影响，Python标准库处理正确

### 总结: 所有问题都已解决，没有遗留问题。

### 6.4 单次查询

```bash

```

### 6.5 编程使用

```python
import asyncio
from claude_agent.agent import create_agent

async def main():
    agent = create_agent(
        model="claude-3-5-sonnet-20241022",
        api_key="your_api_key",
    )
    
    state = agent.get_initial_state("Hello, Claude!")
    final_state = await agent.run(state)
    response = agent.get_final_response(final_state)
    print(response)

if __name__ == "__main__":
    asyncio.run(main())
```

### 6.6 添加自定义工具

```python
from claude_agent.tools.base import BaseTool, ToolContext, ToolResult
from claude_agent.state import PermissionResult, PermissionBehavior

class MyCustomTool(BaseTool):
    name = "my_tool"
    description = "Description of my tool"
    input_schema = {
        "type": "object",
        "properties": {
            "param": {
                "type": "string",
                "description": "Parameter description",
            }
        },
        "required": ["param"],
    }
    output_schema = {
        "type": "object",
        "properties": {
            "result": {"type": "string"},
        },
    }
    
    async def check_permissions(self, input, context):
        return PermissionResult(behavior=PermissionBehavior.ALLOW)
    
    async def execute(self, input, context):
        result = do_something(input["param"])
        return ToolResult(str(result), is_error=False)

# 注册
agent.register_tool(MyCustomTool())
```
